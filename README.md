# Odin Recipes

A recipe website built as part of [The Odin Project](https://www.theodinproject.com/) foundations curriculum.

## About the Project
It is a website made for now, only of html. There is a homepage, with multiple links that reffers to the recipes. There are four different recipes with descriptions, a guide and ingredients.
